# Gaby Herbstein

Legacy Gaby Herbstein site archive, preserved with its original photography, typography, and campaign content.

## Structure

- `index.html` - archive entry
- `app/src/` - source assets and styles
- `app/dist/` - compiled output preserved for reference and deployment
- `header.html`, `footer.html` - shared fragments
- `404page.html`, `test.html` - supporting pages preserved from the original site

## Notes

- Keep the artist branding and imagery intact
- Ignore OS metadata, Vercel cache, and editor debris
- Static archive deployment on Vercel
